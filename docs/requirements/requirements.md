# TubeLingo 要件定義書

## アプリ概要

YouTubeの英語動画を使って英語学習ができるWebアプリ。
ニュース・技術解説動画のリスニング力向上を目的とする。

---

## ターゲットユーザー

- 英語ニュース・技術解説動画でリスニング力を鍛えたい学習者
- 字幕を見ながら動画を理解したい中級者
- 自分のAnthropicアカウントを持っているユーザー（APIキー自己用意）

---

## 解決する課題

| 課題 | 本アプリの対応 |
|---|---|
| 動画を止めて単語を調べるのが面倒 | 字幕テキストを選択するだけで即時翻訳 |
| 単語の意味は分かっても文脈が理解できない | 前後の字幕を含めてAIが解説 |

---

## 機能一覧

### MVP（v1.0）

| # | 機能 | 概要 |
|---|---|---|
| F-01 | APIキー入力 | AnthropicのAPIキーをメモリ上に保持（ページを閉じると消去） |
| F-02 | YouTube URL入力 | URLから動画IDを抽出して字幕取得を開始 |
| F-03 | 字幕取得・表示 | `youtube-transcript` で字幕を取得して表示 |
| F-04 | 字幕同期 | 動画の再生位置に合わせて現在字幕をハイライト・自動スクロール |
| F-05 | テキスト選択 → 翻訳 | 選択テキストをMyMemory APIで即時日本語訳 |
| F-06 | テキスト選択 → AI解説 | 選択テキスト＋前後字幕をClaudeに送り詳細解説 |
| F-07 | AIチャット継続質問 | 同一字幕ブロックに対して追加質問が可能 |
| F-08 | AI質問済みマーク | 質問済み字幕ブロックに💬アイコンを表示 |

### 将来対応（v2.0以降）

| # | 機能 |
|---|---|
| F-10 | メモ機能 |
| F-11 | 動画についてのAI会話機能 |

## 非機能要件

| 項目 | 要件 |
|---|---|
| 対応デバイス | スマホ（縦画面）を基本とし、PCブラウザでも動作すること |
| レスポンス | 翻訳は2秒以内、AIストリーミングは開始まで3秒以内 |
| セキュリティ | APIキーはメモリ上のみに保持し、localStorage・サーバー・ログに一切残さないこと |
| 可用性 | Vercelのデプロイ環境で常時アクセス可能 |

---

## 制約・前提

- ユーザーは自身のAnthropicアカウントとAPIキーを用意する
- 字幕は対象動画にYouTubeの自動または手動字幕が存在すること（ない場合は非対応としてエラー表示）
- 翻訳機能（MyMemory API）は英語→日本語のみ対応

## エラー定義

| エラーコード | 原因 | ユーザー向けメッセージ |
|---|---|---|
| E-01 | 字幕非対応動画 | 「この動画には字幕がありません」 |
| E-02 | 無効なYouTube URL | 「正しいYouTube URLを入力してください」 |
| E-03 | APIキー未設定または不正 | 「APIキーが設定されていないか無効です。正しいキーを入力してください」 |
| E-04 | ネットワークエラー | 「通信エラーが発生しました。再度お試しください」 |
| E-05 | Claude APIエラー | 「AI機能でエラーが発生しました：(詳細)」 |
