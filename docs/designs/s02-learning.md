## 画面イベント

| 実行条件 | 処理内容 | 備考 |
|---|---|---|
| 初期表示 | YouTubeプレーヤーを即座に初期化・表示する。同時に `/api/transcript?url=...` を呼び出して字幕を取得する。取得中は字幕エリアにローディング表示。 | 動画視聴と字幕取得を並行処理 |
| 字幕取得成功 | 字幕リストを表示し、500ms ごとに再生位置をポーリング開始 | |
| 字幕取得失敗 | 字幕エリアにエラーメッセージを表示する。動画は引き続き視聴可能 | E-01, E-04 |
| 再生位置の更新（500ms ごと） | 現在再生中の字幕ブロックをハイライトし、自動スクロールで画面内に収める | `useTranscriptSync` で算出 |
| 字幕テキストをクリック | テキストをハイライトし、再生位置をテキストの位置に変更する。 |  |
| 字幕テキストをドラッグ選択 | 選択範囲の上部に「翻訳」ボタンと「AIに聞く」ボタンを表示。 | 選択解除で非表示 |
| 「翻訳」ボタン押下 | MyMemory APIを実行し翻訳結果をポップアップで表示する。 | |
| 翻訳API呼び出し失敗 | ポップアップにエラーメッセージを表示する | E-04 |
| 「AIに聞く」ボタン押下 | S-02-M AIチャットモーダルを開く（選択テキストを引き継ぐ） | |
| ChatIconタップ | S-02-M AIチャットモーダルを開く（保存済みチャット履歴を復元して表示） | |
