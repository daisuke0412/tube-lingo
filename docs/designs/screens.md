## 画面遷移図

```
[S-01 初期画面]
    │
    │ 開始ボタン押下
    │
    ├─ 字幕取得成功 ──→ [S-02 学習画面]
    │                       │
    │                       ├─ [AIに聞く] / 💬タップ ──→ [S-02-M AIチャットモーダル]
    │                       │                                    │
    │                       │                          ✕ボタン ──┘（S-02に戻る）
    │                       │
    │                       └─ APIエラー ──→ 各画面内にインライン表示
    │
    └─ 字幕取得失敗 ──→ S-01 内にインライン表示
```

## 各画面の共通フォーマット

各画面設計書は以下のセクションで構成する。

| セクション | 内容 |
|---|---|
| ヘッダー表 | 画面ID / 画面名 / 表示条件 / 関連機能 |
| モックアップ | ASCIIアートによるUI構成。状態ごとに複数記載可 |
| 入力バリデーション | フィールドごとのルール。なければ「なし」と記載 |
| 画面イベント | 実行条件・処理内容・備考（初期表示を必ず含む） |
