## 画面イベント

### 初期表示

YouTubeプレーヤーを即座に初期化・表示する。同時に `/api/transcript?url=...` を呼び出して字幕を取得する。
動画視聴と字幕取得は並行処理する。字幕取得中は字幕エリアにローディングを表示する。

### 字幕取得成功

字幕リストを表示し、500ms ごとに再生位置のポーリングを開始する。

### 字幕取得失敗（字幕なし）

字幕エリアに「この動画には字幕がありません」を表示する。動画は引き続き視聴可能。

### 字幕取得失敗（通信エラー）

字幕エリアに「字幕の取得に失敗しました。しばらく経ってから再試行してください」を表示する。動画は引き続き視聴可能。

### 再生位置の更新（500ms ごと）

現在再生中の字幕ブロックをハイライトし、自動スクロールで画面内に収める。`useTranscriptSync` で算出する。

### ▶ アイコンをクリック

その字幕ブロックの開始時刻に再生位置を移動する。

### 字幕テキストをドラッグ選択

選択範囲の上部に「翻訳」ボタンと「AI質問」ボタンを表示する。選択解除で非表示になる。字幕テキストをクリックしても何も起きない。

### 「翻訳」ボタン押下

MyMemory API を呼び出し、翻訳結果をポップアップで表示する。

### 翻訳API呼び出し失敗

ポップアップに「翻訳に失敗しました。しばらく経ってから再試行してください」を表示する。

### 「AIに聞く」ボタン押下

S-02-M AIチャットモーダルを開く。選択テキストを引き継いで渡す。

### ChatIcon タップ

S-02-M AIチャットモーダルを開く。メモリ `state` に保存済みのチャット履歴を復元して表示する。
