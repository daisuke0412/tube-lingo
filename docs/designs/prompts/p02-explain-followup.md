# P-02 追加質問プロンプト

| 項目 | 内容 |
|---|---|
| プロンプト ID | P-02 |
| プロンプト名 | 追加質問 |

---

## システムプロンプト

```
あなたは英語学習のアシスタントです。
ユーザーの質問に日本語で回答してください。
マークダウン記法（#・**・- 等）は使わず plain text で出力してください。
回答は最大3文にまとめてください。
```

## messages 配列

`chat_history` 全件 + 今回の `user_message` を末尾に追加して渡す。

```json
[
  { "role": "user", "content": "「photosynthesis」について解説してください。" },
  { "role": "assistant", "content": "翻訳: 光合成\n\n解説: ..." },
  { "role": "user", "content": "{{user_message}}" }
]
```

---

## パラメータ

| パラメータ | 型 | 説明 | 埋め込み箇所 |
|---|---|---|---|
| `chat_history` | `ChatMessage[]` | 過去のやり取り全件（初回解説〜直前のやり取りまで） | messages配列（先頭〜末尾から2番目） |
| `user_message` | `string` | 今回のユーザーメッセージ | messages配列（末尾） |

---

## 出力フォーマット例

```
光合成は葉の中の葉緑体という器官で行われます。葉緑体が光エネルギーを使って二酸化炭素と水を糖に変換します。
```
