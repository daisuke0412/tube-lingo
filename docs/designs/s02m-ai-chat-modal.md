## 画面イベント

| 実行条件 | 処理内容 | 備考 |
|---|---|---|
| 初期表示（新規） | 選択テキスト＋前後3行の字幕を含めて `/api/explain` を呼び出し、AIがストリーミング回答を開始する | 入力欄・送信ボタンを無効化。回答完了後に有効化 |
| 初期表示（APIキー未設定） | チャット欄にエラーメッセージ（E-03）を表示し、入力欄・送信ボタンを無効化する | API呼び出しは行わない |
| 初期表示（履歴復元） | メモリ `state` から保存済みチャット履歴を読み込み表示する | API呼び出しは行わない。入力欄はすぐにアクティブ |
| AI回答のストリーミング受信中 | 回答テキストをリアルタイムでチャット欄に追記表示する | 入力欄・送信ボタンは無効のまま |
| 送信ボタン押下（追加質問） | 入力テキストをユーザーメッセージとして追加し、`/api/explain` を呼び出してストリーミング回答を表示 | 送信後に入力欄をクリア。AI回答完了まで無効化 |
| ✕ボタン押下 | モーダルを閉じ S-02 に戻る。現在のチャット履歴を  メモリ `state` に保存する | 💬 アイコンの表示状態を更新 |
| API呼び出し失敗 | チャット欄にエラーメッセージを表示し、入力欄を有効化する | E-05 / E-04 |
