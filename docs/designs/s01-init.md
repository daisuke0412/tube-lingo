## 入力バリデーション

| フィールド | ルール |
|---|---|
| Anthropic API Key | `sk-ant-` から始まる文字列。任意項目だが、ない場合はAIチャットは利用できない |
| YouTube URL | `youtube.com/watch?v=` または `youtu.be/` の形式 |

---

## 画面イベント

| 実行条件 | 処理内容 | 備考 |
|---|---|---|
| 初期表示 | `tubelingo_last_url` を localStorage から読み込み、YouTube URL フィールドに自動入力する | APIキーは自動入力しない |
| 開始ボタン押下 | バリデーション実施 → `/api/transcript` を呼び出し字幕取得 → 成功時に S-02 へ遷移 | ボタンをローディングスピナーに変更、入力欄を無効化 |
| 開始ボタン押下（字幕取得失敗） | ボタン下にエラーメッセージを赤字インライン表示 | 入力欄・ボタンを再度有効化 |
| 開始ボタン押下（バリデーションエラー） | 該当フィールド下にエラーメッセージを表示 | API呼び出しは行わない |
